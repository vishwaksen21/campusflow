# CampusFlow

CampusFlow is an AI-powered student productivity system that integrates WhatsApp reminders, Google Calendar, and AI summaries using automation (n8n).

## ⚙️ Tech Stack

*   **Backend:** Node.js + Express
*   **Database:** Supabase (PostgreSQL)
*   **Automation:** n8n (Webhook-based logic)
*   **Messaging:** Twilio WhatsApp Sandbox (Free tier)
*   **AI Summary:** OpenAI (ChatGPT) / Groq API (Free tier ready)
*   **Calendar:** Google Calendar API (via n8n OAuth2)

## ✅ Setup Guide (Step-by-step)

### 1. Supabase Database Setup
1. Go to [Supabase](https://supabase.com/).
2. Create a new project.
3. Once created, go to the SQL Editor and run the following query:
   ```sql
   create table users (
     id uuid default uuid_generate_v4() primary key,
     name text not null,
     phone text not null,
     email text not null,
     created_at timestamp with time zone default timezone('utc'::text, now())
   );
   ```
4. Copy your `Project URL` and `anon public` key to `.env`.

### 2. Backend Setup
1. Clone the repository and navigate to the root directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` and fill in the missing values.
   ```bash
   cp .env.example .env
   ```
4. Start the backend server:
   ```bash
   node src/index.js
   ```
   *The server runs on http://localhost:3000.*

### 3. n8n Automation Setup (Local/Desktop)
1. Download and install [n8n Desktop](https://n8n.io/) or run via Docker.
2. Start n8n.
3. Create a new workflow.
4. Click the options menu in the top right -> `Import from File`. Select `n8n/campusflow_workflow.json` from this repository.
5. Setup the nodes:
   *   **Webhook url:** Update your backend's `.env` file (`N8N_WEBHOOK_URL`) with the webhook `Test` or `Production` URL generated by n8n.
   *   **Google Calendar OAuth2:** Open the Google Calendar node, add a new credential, and follow the OAuth2 setup utilizing Google Cloud Console (ensure Calendar API is enabled).
   *   **Twilio WhatsApp Setup:** Go to [Twilio Console](https://www.twilio.com/console) > Messaging > Try it out > Send a WhatsApp message. Join the sandbox using the provided code. In n8n, input your Account SID and Auth Token to configure Twilio credentials. Update the `from` phone number in both Twilio nodes to the Twilio Sandbox WhatsApp number (e.g., `whatsapp:+14155238886`).
   *   **OpenAI Setup:** Go to the OpenAI node and provide your API Key.
6. Make sure to set the workflow to **Active**!

## 🧪 Testing Guide (Using Postman)

### 1. Register User (`POST /register`)
*   **URL:** `http://localhost:3000/register`
*   **Headers:** `Content-Type: application/json`
*   **Body:**
```json
{
  "name": "John Doe",
  "phone": "whatsapp:+1234567890",
  "email": "student@gmail.com"
}
```

### 2. Add Task (Deadline) (`POST /add-task`)
*   **URL:** `http://localhost:3000/add-task`
*   **Headers:** `Content-Type: application/json`
*   **Body:**

{
